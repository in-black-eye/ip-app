# ip-app

Приложение позволяет получать IP адрес через различные API.

Поддерживаемые API: `ip-api` и `jsonip`

Выбираются через переменную окружения **TYPE**.

Команда для выбора `ip-api`:
```
export TYPE=ip-api
```

Команда для выбора `jsonip`:
```
export TYPE=jsonip
```

Запускается на http://localhost:5000
